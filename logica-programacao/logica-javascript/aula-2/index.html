Operadores Aritméticos

  • + Adição
  • - Subtração
  • * Multiplicação
  • / Divisão
  • % Resto da Divisão

Operadores de Comparação

  • = Atribuição
  • == Verifica se o valor é igual
  • === Verifica se o valor e o tipo são iguais (Estritamente Igual)
  • != Verifica se os valores são diferentes
  • !== Verifica se o valor e o tipo são diferentes
  • > Maior que
  • >= Maior ou igual
  • < Menor que
  • < Menor ou igual
  • Em valores binérios, 0 é falso (false) e 1 é verdadeiro (true)

Operadores Lógicos

Lida com Valores e Resultados - Booleans
  • && - (and, E) Seu resultado é verdadeiro se ambos os valores forem verdadeiros (true)
  • || - (or, ou) Seu resultado é verdadeiro se, ao menos, um dos valores for verdadeiro (true)
  • ! - (not, Não) Inverte o valor que é dado a ele
