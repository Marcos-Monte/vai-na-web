Objetos

Objetos são estruturas de dados que nos permitem armazenar dados e funcionalidades utilizando {chaves}

  • Os objetos não garantem uma ordem específica das propriedades. Não são ordenados

  • Pode contar multiplos tipos de dados
  • Utilizamos chaves {} para a sua declaração
  • As propriedades/elementos, são acessados usando 'ponto'

Array

Array são estruturas semalhantes a listas e podem conter diversos tipos de dados. Inclusive, podemos guardar objetos dentro de um array.

  • Os elementos em um array são ordenados e cada um tem um indice numerico. Começando do zero [0, 1, 2...]
  • Pode conter multiplos tipos de dados.
  • Utiliza [ colchetes ] na sua declaração.
  • Os elementos são acessados usando o indice

Métodos de Array

  • Push - Adiciona um elemento no final do Array
  • Unshift - Adiciona um elemento no inicio do Array
  • Pop - Remove o ultimo elemento do Array
  • Shift - Remove o primeiro elemento do Array
  • Length - Retorna o numero de elementos que existe dentro de um Array
  • Sort - Ordena os elementos de um array (crescente ou decrescente)
